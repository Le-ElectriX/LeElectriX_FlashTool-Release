# LeElectriX_FlashTool-Release
Le-ElectriX 刷写工具发布仓库

---

## 更新说明

### v1.1.2

#### 新增功能

- 将运行情况保存到日志文件。
- 输入一次性密码解锁高级功能。
- 扫描功能需要解锁后使用。

#### 优化改善

- 刷写过程失败时提示失败原因和建议操作。
- 解决 EPS650 产品在刷写过程容易失败的问题。
  - 注意：这是由于 EPS650 产品不能在协议规定的时间内给出响应，且没有返回 NRC78 响应导致的，上位机为兼容此问题做出的妥协性调整。
